EYE CONTROLLED ASSISTIVE SYSTEM 

Overview 
An assistive technology solution developed for paralyzed individuals that enables interaction through eye movement tracking.

Features :
Eye-controlled cursor movement
Hands-free interaction
News reading functionality of trending top news in the following languages : 
Tamil
Hindi
Malayalam
Streamlit-based web interface


Technologies Used :
Python
OpenCV
MediaPipe
Streamlit
Text-to-Speech


Future Improvements : 
Better eye-tracking accuracy
Voice assistant integration
Smart home control
Emergency assistance module
